# Compound Interest Calculator

A C++ console application that calculates compound interest over time, showing year-by-year growth with and without monthly deposits.

## Features
- User inputs initial investment, monthly deposit, annual interest rate, and number of years
- Displays two reports: growth with monthly deposits and growth without
- Calculates compounded interest monthly

## Files
- `main.cpp` — Entry point, handles user input and menu
- `InvestmentCalculator.h` — Class declaration
- `InvestmentCalculator.cpp` — Class implementation with calculation logic

## How to Run
Compile with any C++ compiler:
g++ main.cpp InvestmentCalculator.cpp -o calculator
./calculator
